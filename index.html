Specific Objectives:

SELECT userid, username, fullname, age, contact, full_address, position, user_image, userid_image

UPDATE user

UPDATE user

SELECT userid, username, position

  • Admin mechanism to verify card authenticity (This is a conceptual query, actual verification would involve external processes or UI actions):

SELECT c.cardid, c.cardname, c.cardset, c.`condition`, u.username AS owner_username, c.card_image, c.card_hash
FROM card c
JOIN user u ON c.ownerid = u.userid

  • Retrieve all active listings with card and seller details:
    • 
SELECT
    l.listingid,
    c.cardname,
    c.cardset,
    c.`condition`,
    l.price,
    l.listing_type,
    l.status,
    u.username AS seller_username
FROM listing l
JOIN card c ON l.cardid = c.cardid
JOIN user u ON l.sellerid = u.userid
WHERE l.status = 'active';

    Auction Functionality
  • Create an auction listing:
    • 
INSERT INTO listing (cardid, sellerid, price, listing_type, status, created_at)
VALUES (1, 1, 0.00, 'auction', 'active', NOW());

  • Initiate an auction for a listing:
    • 
INSERT INTO auction (listingid, start_time, end_time, starting_price, current_price)
VALUES (2, NOW(), DATE_ADD(NOW(), INTERVAL 7 DAY), 10.00, 10.00);

  • Retrieve active auctions with current prices:
    • 
SELECT
    a.auctionid,
    c.cardname,
    l.listingid,
    a.start_time,
    a.end_time,
    a.starting_price,
    a.current_price,
    u.username AS seller_username
FROM auction a
JOIN listing l ON a.listingid = l.listingid
JOIN card c ON l.cardid = c.cardid
JOIN user u ON l.sellerid = u.userid
WHERE l.status = 'active' AND l.listing_type = 'auction';


    Bidding Management
  • Place a new bid on an auction:
    • 
INSERT INTO bid (auctionid, bidderid, amount, bid_time)
VALUES (1, 1, 15.00, NOW());

  • Update the current price of an auction after a new bid:
    • 
UPDATE auction
SET current_price = (SELECT MAX(amount) FROM bid WHERE auctionid = 1) 
WHERE auctionid = 1;

  • Retrieve all bids for a specific auction:
    • 
SELECT
    b.bidid,
    u.username AS bidder_username,
    b.amount,
    b.bid_time
FROM bid b
JOIN user u ON b.bidderid = u.userid
WHERE b.auctionid = 1
ORDER BY b.bid_time DESC;


    Transaction Review System
  • Place a new bid on an auction:
    • 
INSERT INTO review (transactionid, reviewerid, rating, comment, created_at)
VALUES (1, 2, 5, 'Great seller, card arrived quickly and as described!', NOW());

  • Retrieve all reviews for a specific user (as a seller):
    • 
SELECT
    r.reviewid,
    t.transactionid,
    u_reviewer.username AS reviewer_username,
    r.rating,
    r.comment,
    r.created_at
FROM review r
JOIN transaction t ON r.transactionid = t.transactionid
JOIN user u_reviewer ON r.reviewerid = u_reviewer.userid
WHERE t.sellerid = 1;

  • Calculate average rating for a seller:
    • 
SELECT AVG(r.rating) AS average_rating
FROM review r
JOIN transaction t ON r.transactionid = t.transactionid
WHERE t.sellerid = 1;
